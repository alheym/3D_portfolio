# Portfolio

Portfolio site to showcase completed projects, both educational and pet-projects.
Implemented using react and the three.js library for working with 3D objects on the page.
The page features dynamic elements, project preview cards, and describes the entire learning experience. Also added a feedback form for sending emails.
